#  Marketing and Product Performance Analysis

This project analyzes a marketing and product performance dataset using Exploratory Data Analysis (EDA) and Machine Learning techniques to understand key business insights and predict revenue.

Dataset:  
https://www.kaggle.com/datasets/imranalishahh/marketing-and-product-performance-dataset

---

# Project Objective

The objective of this project is to analyze marketing campaign data to:
- Understand factors affecting revenue generation
- Study customer behavior and engagement
- Evaluate effectiveness of discounts and pricing strategies
- Build a machine learning model to predict revenue

---

#  Dataset Description

The dataset includes marketing campaign and product performance data with features such as:

- Budget
- Clicks
- Conversions
- Revenue Generated
- ROI
- Discount Level
- Units Sold
- Subscription Tier
- Customer Satisfaction
- Bundle Pricing

---

# Research Questions (RQs)

1. How does marketing budget affect revenue generation?
2. What is the relationship between clicks and conversions?
3. How does discount level influence units sold?
4. Does subscription tier affect customer satisfaction?
5. What is the distribution of ROI across campaigns?
6. How does bundle pricing impact sales?
7. Can we predict revenue using marketing features?

---

# Methodology

The project follows these steps:

1. Data loading and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Data visualization (scatter plots, box plots, histograms)  
4. Statistical analysis  
5. Machine learning model building  
6. Model evaluation  

---

# Machine Learning Model

- Model Used: Random Forest Regressor  
- Target Variable: Revenue Generated  
- Evaluation Metric: Mean Squared Error (MSE)

---

# Outputs

The project generates:

- 7 research-based visualizations (PDF files)
- 7 summary tables (CSV files)
- Correlation analysis
- Actual vs Predicted revenue comparison
